# Understanding and Using Generative AI (Series 1 - Non-Technical)

## Overview

A six-week seminar series designed for faculty, staff, and students who want to understand and effectively use generative AI tools in their work. **No coding experience required.**

---

## Series Goals

By the end of this series, participants will be able to:
- Understand how large language models work (conceptually)
- Choose the right AI model for different tasks
- Write effective prompts to get better results
- Use AI responsibly and ethically
- Leverage advanced tools like NotebookLM and Custom GPTs
- Build AI-assisted workflows for their work

---

## Course Structure

### Week 1: Introduction to Generative AI
**Topics:** What are LLMs? How do they work? Current landscape
**Hands-On:** Create accounts, first experiments
**Assignment:** Compare ChatGPT and Claude

### Week 2: Model Differences and Selection
**Topics:** ChatGPT vs Claude vs Gemini, context windows, strengths
**Hands-On:** Comparative testing across models
**Assignment:** Real-world model testing

### Week 3: Prompt Engineering Fundamentals
**Topics:** Core principles, common patterns, iteration
**Hands-On:** Improve prompts, practice patterns
**Assignment:** Build personal prompt library

### Week 4: Advanced Prompting and Responsible Use
**Topics:** Chain-of-thought reasoning, hallucinations, red-team auditing, ethics, bias, privacy
**Hands-On:** Verification strategies (CoVe pattern), bias detection, red-team auditing
**Assignment:** Logic audit using 3-step verification pipeline

### Week 5: Beyond Chat - NotebookLM and Custom GPTs
**Topics:** Grounded AI, reducing hallucinations, RAG (Retrieval-Augmented Generation), building knowledge agents
**Hands-On:** Create NotebookLM for document analysis, design Custom GPT system instructions
**Assignment:** Create personalized knowledge agent for weekly task

### Week 6: Orchestration & Building Your AI Workflow
**Topics:** Multi-tool workflows, Scite for research verification, agentic AI in healthcare, workflow integration
**Hands-On:** Explore Scite, design complete workflow pipeline, integrate weeks 1-5 concepts
**Assignment:** Create personal AI workflow map for recurring task

## Frequently Asked Questions

**Q: Do I need to know how to code?**
A: No! This series requires no coding experience.

**Q: How much will this cost?**
A: Free tiers of all tools are sufficient. Optional paid tiers (~$20/month) add features but aren't required.

**Q: What if I miss a session?**
A: Materials provided; recordings available if sessions recorded. Catch up before next week.

**Q: Can I use this for sensitive work?**
A: With proper precautions (de-identification, privacy awareness). We cover this Week 4.

**Q: Will AI replace my job?**
A: These tools augment human capabilities, not replace human judgment. Your expertise remains essential.

**Q: How fast is the field changing?**
A: Very fast! We focus on principles that transfer across tools and updates.
